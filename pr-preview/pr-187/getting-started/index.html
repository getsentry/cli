Installation

Package Managers

Install globally with your preferred package manager:

npm install -g sentry
pnpm add -g sentry
yarn global add sentry
bun add -g sentry

Or run directly without installing:

npx sentry --help
pnpm dlx sentry --help
yarn dlx sentry --help
bunx sentry --help
