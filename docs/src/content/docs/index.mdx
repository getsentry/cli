<div class="quick-start">

## Quick Start

```bash
# Authenticate (opens browser, handles everything)
sentry auth login

# View an issue — short IDs just work
sentry issue view WQ

# Get AI-powered root cause analysis
sentry issue explain WQ

# List issues — project auto-detected from your code
sentry issue list
```
